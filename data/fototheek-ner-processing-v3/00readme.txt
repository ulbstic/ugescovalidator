README
------

This folder contains the NER processed files for the Ugesco project.

The folder output-monalisa-v2 contains the JSON files for the subset
of 5 images, listed in mona-lisa-v2.txt.

As requested, there is now a json file for the LEGEND field (photo)
and another one for the BPALLAS(NF) (group) selection.

Note that the five images are a subset of the other
ugc-ner*photo*.json files.

The JSON files have been validated with the Ugesco json schema.
